District Clustering

This app clusters NYC Election Districts on two variables using K-Means clustering. First, select two variables from the drop-down menus below. The graph will update showing the clusters using a default setting of 3. Then, adjust the number of clusters to any number between 1-9. The graph will show how the data splits between these clusters, and the table below the graph will show the mean values of the selected columns for each cluster. Once you are happy with the number of clusters and the division of the data, scroll down to the map to see where the clusters lie geographically.

NB: all data has been standardized, so 0 is the mean for each variable.
